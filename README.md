# Controle de Almoxarifado

Aplicativo mobile desenvolvido em React Native para controle de estoque de materiais de enfermagem.

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
- MockAPI

## Funcionalidades da Sprint 1

- Cadastro de materiais
- Listagem dinâmica de materiais
- Consumo de API REST com GET
- Cadastro de materiais utilizando POST
- Atualização automática da lista após cadastro

## Como executar

```bash
npm install
npm start
